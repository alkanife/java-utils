# jCommander utils
A collection of utility classes for jCommander.

**Add to your pom.xml:**
````xml
<repository>
    <id>alka-repo</id>
    <url>https://repo.alka.dev/repository/maven-public/</url>
</repository>

<dependency>
    <groupId>dev.alka</groupId>
    <artifactId>jcommander-utils</artifactId>
    <version>VERSION</version>
</dependency>
````
